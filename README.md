# HumToNotes Website (GitHub Pages)

This repo hosts the HumToNotes marketing + support website for GitHub Pages.

## Pages

- `/` — marketing landing page (`index.html`)
- `/support.html` — Support URL for App Store Connect
- `/privacy.html` — Privacy Policy URL for App Store Connect

## Publish on GitHub Pages

In GitHub:

1. **Settings → Pages**
2. **Build and deployment**
   - Source: **Deploy from a branch**
   - Branch: **main** / **(root)**

After it deploys, your site will be available at:

- `https://humtonotes.github.io/` (or your configured custom domain)

## Customize

Update these placeholders:

- Support email in `support.html` (`support@humtonotes.com`)
- Privacy email in `privacy.html` (`privacy@humtonotes.com`)
- Any privacy wording to match your actual app behavior
